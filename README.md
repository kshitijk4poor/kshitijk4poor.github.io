# Kshitij Kapoor's Portfolio

This is the personal portfolio website of Kshitij Kapoor, a 19-year-old self-taught developer with interests in cryptography, cybersecurity, deep learning, and distributed systems.

## Features

- Simple and clean design using Pico CSS
- Responsive layout
- Social media links (LinkedIn, GitHub, Blog)

## Technologies Used

- HTML5
- CSS3
- JavaScript
- [Pico CSS](https://picocss.com/) for styling
- [Font Awesome](https://fontawesome.com/) for icons
